# Clojure Turbo

## Installation

### Versions

* [turbo versions](http://repo.clojars.org/kwrooijen/turbo/0.0.1-SNAPSHOT/)

### project.clj

```clojure
:dependencies [[kwrooijen/turbo "0.0.1-xxxxxxxx.yyyyyy-z"]
;; Or lastest: [kwrooijen/turbo "0.0.1-SNAPSHOT"]
               ,,,]
```

### package.json

```javascript
{
    "dependencies": {
        "@hotwired/turbo": "^7.0.0-beta.2"
    }
}
```
